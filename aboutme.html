About Me

is it obvious i crammed this within the last three hours

Hey, I'm Lana! This website is the embodiment of all my cringy childhood memories. To be clear, I'm not ashamed of the fact that I like shoujo manga/anime NOW--why would I? A lot have some of the most beautiful tales I've ever read! But I was "not like other girls" back then, so I thought it was lame that such a "cool" girl like me would be into something so girly. Kinda dumb, I know. This website is sort of an apology for that?

Honestly, I'm not that satisfied on how this turned out. There was so much I wanted to do, and I only have myself to blame for cramming this project TT

Oh well, gotta do better next time--thank you so much for reading!
